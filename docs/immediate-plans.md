## Phase 2: Dev Install Command

After scaffolding, create project-level command in mind-games-mg's mg-state:

**`mg dev install`** - Generates and installs shell script pointing to current repo:

```bash
#!/bin/bash
# Generated by mg dev install
# Repo: {{ repo_path }}
exec uv run --project {{ repo_path }}/worktrees/main/mg-cli mg "$@"
```

- Detects repo location at install time
- Installs to `~/.local/bin/mg` (respecting XDG/user preference)
- Works in interactive and non-interactive shells

**Workflow:**
- `mg` runs from main worktree (stable)
- Develop features on branches, merge to main when tested
- Changes available immediately after merge
